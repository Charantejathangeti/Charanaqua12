Charan Aquarium - Full Website

Quick start (development)

1. Install dependencies: `npm install`
2. Run server: `npm run dev` (requires nodemon) or `npm start`
3. Open http://localhost:3000


Notes:
- Admin login is protected by a simple env password (see server.js). For production replace with proper auth.
- OTP is simulated: server prints OTP to console for development.
- WhatsApp checkout uses wa.me link to 6302382280.
